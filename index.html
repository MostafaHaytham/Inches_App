GITHUB PAGES SETUP
Deploy this prototype suite publicly in under 5 minutes — free, permanent link
Step 01
Create GitHub Repo
Go to github.com → sign in or create a free account → click the + button → New repository.

Name it inches-prototype and set visibility to Public. Leave all other options as default. Click Create repository.
Step 02
Upload the 4 Files
On your new repo page, click Add file → Upload files.

Drag and drop all 4 files at once:
index.html
INCHES_Athlete_v3.html
INCHES_Coach_v4.html
INCHES_Admin_v3.html
Click Commit changes.
Step 03
Enable GitHub Pages
In your repo, go to Settings → Pages (left sidebar).

Under Source, select:
Branch: main
Folder: / (root)
Click Save. GitHub will build and deploy — takes about 60 seconds.
Step 04
Get Your Live URL
After saving, your live URL appears at the top of the Pages settings:
# Your permanent public URL:
https://[username].github.io
/inches-prototype
Share this link with anyone — works on iPhone, iPad, desktop. No account needed to view.
Step 05
Add to iPhone Home Screen
Open the URL in Safari on iPhone.

Tap the Share button (box with arrow) → Add to Home Screen → name it INCHES → tap Add.

It will appear on your home screen and open full-screen like a native app.
Step 06
Update Anytime
To update any file, go back to your repo → click the file → click the pencil (edit) icon → paste the new content → Commit changes.

Or drag a new file with the same name over the old one using Add file → Upload files. GitHub Pages auto-redeploys in ~60 seconds.
